# Task Manager (Python) - Projeto de Portfólio

Gerenciador de tarefas simples em linha de comando, criado para demonstrar lógica, manipulação de arquivos e organização de código em Python.

## Funcionalidades
- Listar tarefas
- Adicionar tarefa
- Remover tarefa por índice
- Apagar todas as tarefas
- Persiste tarefas em `tasks.txt`

## Como executar (no celular)
### Opção 1 — Usando Pydroid 3 (Android)
1. Instale o app **Pydroid 3** na Play Store.
2. Crie um novo arquivo chamado `task_manager.py` e cole o código.
3. Execute pelo botão “Run”. O arquivo `tasks.txt` será criado automaticamente.

### Opção 2 — Usando Termux (Android)
1. Instale **Termux**.
2. Instale Python: `pkg install python`
3. Faça upload do `task_manager.py` para a pasta do Termux (via `termux-setup-storage` e movendo o arquivo).
4. Rode: `python task_manager.py`

### Opção 3 — No PC
1. Certifique-se de ter Python 3 instalado.
2. Rode: `python task_manager.py`

## Estrutura
- `task_manager.py` — código principal
- `tasks.txt` — arquivo criado/atualizado pelo programa
- `README.md` — instruções
